# Web to Podcast

Convert web content to podcast audio.

## Setup
1. `pip install -r requirements.txt`
2. Add your OpenAI API key to `.env`: `OPENAI_API_KEY=your_key`
3. `python app.py`
4. Open http://localhost:5000

Supports URLs and PDFs.
